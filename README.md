# FormFlow

FormFlow is a backend system designed to simulate a simple form submission and user management platform. It is built to demonstrate real-world backend engineering skills using Spring Boot and PostgreSQL.

The project focuses on clean architecture, REST API design, and database integration — forming the foundation for scalable backend systems and SaaS-style applications.

---

## 🚀 Features

- User management system (Create & Retrieve users)
- RESTful API built with Spring Boot
- PostgreSQL database integration
- Layered architecture (Controller → Service → Repository)
- JPA-based ORM data persistence
- Clean and extensible backend structure

---

## 🧱 Tech Stack

- Java 17+ / 21+
- Spring Boot
- Spring Web (REST API)
- Spring Data JPA
- PostgreSQL
- Maven
